# Estructura de carpetas para ML-TN-Sync

```
ml-tn-sync/                      # Carpeta raíz del proyecto
│
├── src/                         # Código fuente
│   ├── __init__.py              # Hace que src sea un paquete Python
│   ├── api/                     # Módulos de conexión a APIs
│   │   ├── __init__.py
│   │   ├── mercadolibre.py      # Clase para API de Mercado Libre
│   │   └── tiendanube.py        # Clase para API de Tienda Nube
│   │
│   ├── core/                    # Lógica principal
│   │   ├── __init__.py
│   │   ├── synchronizer.py      # Clase sincronizadora
│   │   └── price_calculator.py  # Lógica de cálculo de precios
│   │
│   └── utils/                   # Utilidades
│       ├── __init__.py
│       └── logger.py            # Configuración de logging
│
├── config/                      # Archivos de configuración
│   ├── __init__.py
│   ├── settings.py              # Configuración general
│   └── credentials.example.json # Plantilla de credenciales (para GitHub)
│
├── scripts/                     # Scripts de instalación y ejecución
│   ├── setup.sh                 # Script de instalación para Linux/Mac
│   ├── setup.bat                # Script de instalación para Windows
│   ├── run.sh                   # Script de ejecución para Linux/Mac
│   └── run.bat                  # Script de ejecución para Windows
│
├── logs/                        # Directorio para archivos de registro
│   └── .gitkeep                 # Mantener carpeta vacía en git
│
├── tests/                       # Pruebas unitarias
│   ├── __init__.py
│   ├── test_mercadolibre.py
│   └── test_tiendanube.py
│
├── .gitignore                   # Archivos a ignorar en git
├── README.md                    # Documentación principal
├── requirements.txt             # Dependencias del proyecto
└── main.py                      # Punto de entrada principal
```

## Archivos importantes

### main.py
Este es el punto de entrada principal del script que orquesta todo el proceso.

### config/credentials.example.json
```json
{
    "mercadolibre": {
        "client_id": "TU_CLIENT_ID_MERCADOLIBRE",
        "client_secret": "TU_CLIENT_SECRET_MERCADOLIBRE",
        "refresh_token": "TU_REFRESH_TOKEN_MERCADOLIBRE",
        "user_id": "TU_USER_ID_MERCADOLIBRE"
    },
    "tiendanube": {
        "api_key": "TU_API_KEY_TIENDANUBE",
        "user_id": "TU_USER_ID_TIENDANUBE"
    }
}
```
